# Email Templates Testing

This folder contains HTML test files for previewing email templates in a browser.

## Test Files

1. **verification-email.html** - Email verification template
2. **password-reset.html** - Password reset request template
3. **password-changed.html** - Password changed confirmation template
4. **welcome-email.html** - Welcome email after verification template

## How to Test

1. Open any HTML file in your web browser
2. Check the layout, styling, and content
3. Test responsiveness by resizing the browser window
4. Verify that all elements display correctly

## Features

- **Montserrat Font**: Modern, professional typography
- **LTR Direction**: Left-to-right text layout
- **Blue Gradient Header**: Brand colors (#3678FF to #2563EB)
- **Rounded Corners**: 25px for containers, 12px for elements
- **Responsive Design**: Mobile-friendly layout
- **Modern UI**: Clean, professional appearance

## Notes

- All templates use inline CSS for maximum email client compatibility
- Logo is embedded as base64 SVG (replace with actual logo)
- Templates are English-only
- All links are placeholder (#) - replace with actual URLs in production
